CCInternet API Documentation

POST /register

Registers a new user in the system.

Show Details

Request Payload:

{
    "username": "exampleUser",
    "password": "examplePassword"
}

Response:

{
    "message": "Registration successful",
    "subIP": "192.0.0.1",
    "userID": "3a21edff-92eb-4019-89be-3e15d141625a"
}

ComputerCraft Example:

-- Lua example for registration
local data = {
    username = "exampleUser",
    password = "examplePassword"
}
local response = http.post("https://Your_IPAddr/register", textutils.serializeJSON(data), {["Content-Type"] = "application/json"})
if response then
    print(response.readAll())
    response.close()
else
    print("Failed to contact the server.")
end

POST /send

Sends a message from one user to another based on their Sub-IP. Messages can include simple text or encoded content such as file data.

Show Details

Request Payload:

{
      "fromUserID": "3a21edff-92eb-4019-89be-3e15d141625a",
      "targetSubIP": "192.0.0.2",
      "message": "U29tZSBjb250ZW50IHRvIGJlIHNlbnQ="
  }

Response:

{
      "message": "Message sent successfully"
  }

ComputerCraft Example:

-- Lua example for /send
  local function base64Encode(data)
      -- Base64 encoding function here (omitted for brevity)
  end
  
  local data = {
      fromUserID = "3a21edff-92eb-4019-89be-3e15d141625a",
      targetSubIP = "192.0.0.2",
      message = base64Encode("Some content to be sent")
  }
  local response = http.post("https://Your_IPAddr/send", textutils.serializeJSON(data), {["Content-Type"] = "application/json"})
  if response then
      print(response.readAll())
      response.close()
  else
      print("Failed to contact the server.")
  end

POST /reverse

Simulates a web request (GET or POST) sent to another user's Sub-IP.

Show Details

Request Payload (GET):

{
      "fromUserID": "3a21edff-92eb-4019-89be-3e15d141625a",
      "targetSubIP": "192.0.0.2",
      "method": "GET",
      "path": "file.txt"
  }

Response (GET):

{
      "response": "RklMRTo="
  }

Request Payload (POST):

{
      "fromUserID": "3a21edff-92eb-4019-89be-3e15d141625a",
      "targetSubIP": "192.0.0.2",
      "method": "POST",
      "path": "file.txt",
      "data": "U29tZSBjb250ZW50IGZvciB0aGUgZmlsZQ=="
  }

Response (POST):

{
      "response": "MjAxOiBGaWxlIGNyZWF0ZWQ="
  }

ComputerCraft Example for GET:

-- Lua example for reverse GET
  local data = {
      fromUserID = "3a21edff-92eb-4019-89be-3e15d141625a",
      targetSubIP = "192.0.0.2",
      method = "GET",
      path = "file.txt"
  }
  local response = http.post("https://Your_IPAddr/reverse", textutils.serializeJSON(data), {["Content-Type"] = "application/json"})
  if response then
      print(response.readAll())
      response.close()
  else
      print("Failed to contact the server.")
  end

ComputerCraft Example for POST:

-- Lua example for reverse POST
  local function base64Encode(data)
      -- Base64 encoding function here (omitted for brevity)
  end
  
  local data = {
